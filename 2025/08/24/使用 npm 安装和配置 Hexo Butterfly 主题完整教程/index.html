    • 选择源为 “GitHub Actions” 或对应的分支

常见问题解决

  1. 页面样式错乱：

    • 确保已安装所需渲染器：npm install hexo-renderer-pug hexo-renderer-stylus --save

  2. 配置修改未生效：

    • 执行 hexo clean && hexo generate 重新生成
    • 确保修改的是 _config.butterfly.yml 而不是 node_modules 中的文件

  3. 本地正常但线上异常：

    • 检查 Vercel/GitHub Pages 的构建日志
    • 确保所有依赖已正确添加到 package.json
